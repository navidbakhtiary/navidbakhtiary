# 👋 Hi, I'm Navid Bakhtiary

Backend Engineer with 13+ years of experience building database-driven applications, backend services, and REST APIs using **Laravel**, **PHP**, **PostgreSQL**, **MySQL**, and **Python**.

I am currently developing **AI-enabled decision-support systems** at **Julius Kühn-Institut** in Germany, where I work on backend platforms that integrate environmental sensor data, geospatial datasets, machine learning services, and LLM-powered functionality for scientific and real-world analysis.

My main focus is on:
- Designing clean and scalable backend architectures
- Building maintainable REST APIs and database systems
- Integrating Python, FastAPI, ML, and LLM services into production workflows
- Developing practical software for research, analytics, and decision support
